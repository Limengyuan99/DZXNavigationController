# DZXNavigationController
通过自定义UINavigationController和UIViewController实现一个更加灵活的导航栏。

##特性
* 全屏的转场动画
* 百分比驱动的交互转场
* 方便的导航栏背景透明方法
* 提供类似原生UINavigationController的属性
* 可选择的滑动返回手势方式（无滑动返回/边缘滑动返回/全屏滑动返回）
